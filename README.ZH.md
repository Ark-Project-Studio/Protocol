## 简介

Protocol 是一个用于 Minecraft Bedrock Edition 的协议库，用于处理数据包收发、协议编解码、网络通信等底层逻辑

## 特性

- 支持 Minecraft Bedrock Edition 协议的序列化与反序列化
- 多种数据包类型与自动注册、分发机制
- 跨平台网络通信能力
- 可灵活扩展的数据包处理和事件钩子
- 简单易用的 API 设计，方便接入自定义逻辑
- 良好的模块分层与文档注释
